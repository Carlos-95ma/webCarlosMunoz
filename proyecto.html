Propuesta de surtido dinamico

Python · SQL · SAS — Sistema de reposición y control automatizado

Descripción del Proyecto

Sistema automatizado de control y reposición de inventarios que minimiza roturas de stock y optimiza el surtido por tienda mediante análisis avanzado de datos.

Objetivo

Desarrollar un modelo analítico para la optimización dinámica del surtido, equilibrando cobertura, rotación y disponibilidad sin afectar las ventas.

Metodología

Integración de datos desde tres DataMarts corporativos (stocks, ventas, codificación de referencias). Extracción mediante SAS y modelado con SQL para generar vistas consolidadas.

Se calculan indicadores clave como cobertura, ritmos de venta y rotación, limitando variaciones del surtido a ±20 % respecto a la versión anterior.

Indicadores Clave

  • Cobertura por proveedor, centro y división.
  • Ritmos de venta comparativos (semana, mes, campaña).
  • Variación de stock en unidades y euros.

Resultados

Generación de recomendaciones automáticas semanales que reducen el exceso de inventario y minimizan roturas de stock. Incluye un resumen de impacto económico que compara escenario base vs. optimizado.

Resultado: herramienta escalable y recurrente para la gestión estratégica del stock.
